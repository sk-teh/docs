xpertioPlatformService

xpertioPlatformService is a component where it gets the platform identifier which defines how this device should be provisioned.

Parameters:

  • Name (string) – Component name

  • meta (string) – Component name

  • platformId (string) – Get the platform identifier which defines how this device should be provisioned

  • platformVer (string) – Version of the controller

  • memAvailable (string) – Available memory of the controller
